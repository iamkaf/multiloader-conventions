# Minimal Sample Consumer

This sample is documentation for the intended first adoption shape.

It is not part of the main build yet. The first real consumer should be `multiloader-template`, followed by `mochila2`.

## Files

- `settings.gradle`
- `build.gradle`
- `gradle.properties`

## Development Flow

Use `includeBuild("../../")` while iterating locally on the convention repo. Replace that with published plugin resolution only after the conventions stabilize.
